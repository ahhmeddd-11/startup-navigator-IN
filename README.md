## 📖 Overview

- Startup Navigator centralizes startup guidance into one intelligent platform.
- It combines curated knowledge, AI assistance, government schemes, and startup resources.
- Built as a modern full-stack application using React, Django REST Framework and PostgreSQL.

## 🎯 Problem Statement

- Entrepreneurs spend hours searching multiple websites.
- Information is fragmented and difficult to verify.
- Startup Navigator provides one unified experience.

## ✨ Features

- JWT Authentication
- AI Assistant powered by Gemini
- Knowledge Base
- Government Schemes
- Startup Resources
- Bookmarks
- Search History
- Recommendations
- Admin Dashboard
- Responsive UI

## 🛠 Tech Stack

- Frontend: React, Vite, TypeScript, Tailwind CSS, TanStack Router, TanStack Query, Axios
- Backend: Python, Django, Django REST Framework, SimpleJWT
- Database: PostgreSQL
- AI: Google Gemini
- Deployment: Render + Vercel

## 🏗 Architecture

- React → Axios → Django REST API → Services → PostgreSQL → Gemini API
